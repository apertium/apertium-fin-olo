    Macros

    Macros are helper functions in apertium transfer files.

    remove-semtag1

    Parametres: 1
    1. let tl[1]['a_sem'] ≔ ""

    Rules

    The actual rules concerning stuff.

    REGLA: pron

    Matching pattern:

    1. pron

    Action:

    1. remove-semtag1($1)
    2. let tl[1]['x_func'] ≔ ""
    3. let tl[1]['a_func'] ≔ ""
    4. let tl[1]['a_poss'] ≔ ""
      1. if
        1. tl[1]['a_cas'] ≟ <acc>
        3. then:
        1. let tl[1]['a_cas'] ≔ <par>
    6. Output:
      1. [
        1. tl[1]['whole']
        ]pron <SN>

    REGLA: nom

    Matching pattern:

    1. nom

    Action:

    1. remove-semtag1($1)
    2. let tl[1]['x_func'] ≔ ""
    3. let tl[1]['a_func'] ≔ ""
    4. let tl[1]['a_poss'] ≔ ""
      1. if
        1. tl[1]['a_cas'] ≟ <acc>
        3. then:
        1. let tl[1]['a_cas'] ≔ <gen>
    6. Output:
      1. [
        1. tl[1]['whole']
        ]nom <SN>

    REGLA: verb

    Matching pattern:

    1. verb

    Action:

    1. remove-semtag1($1)
    2. let tl[1]['x_func'] ≔ ""
    3. let tl[1]['a_func'] ≔ ""
    4. let tl[1]['a_act'] ≔ ""
    5. Output:
      1. [
        1. tl[1]['whole']
        ]v <SV>

    REGLA: DEFAULT

    Matching pattern:

    1. default

    Action:

    1. let tl[1]['a_sem'] ≔ ""
    2. let tl[1]['x_func'] ≔ ""
    3. let tl[1]['a_func'] ≔ ""
    4. Output:
      1. [
        1. tl[1]['whole']
        ]default sl[1]['a_func']

    REGLA: DEFAULT QST

    Matching pattern:

    1. default
    2. qst

    Action:

    1. let tl[1]['a_sem'] ≔ ""
    2. let tl[1]['x_func'] ≔ ""
    3. let tl[1]['a_func'] ≔ ""
    4. Output:
      1. [
          ]default sl[1]['a_func']
